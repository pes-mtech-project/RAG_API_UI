# Container Management Scripts

This directory contains scripts for managing Docker containers locally and in production.

## Scripts

- `build.sh` - Build containers locally
- `deploy-local.sh` - Deploy containers locally with Docker Compose
- `deploy-prod.sh` - Deploy containers to production
- `health-check.sh` - Check container health status
- `logs.sh` - View container logs
- `cleanup.sh` - Clean up old containers and images
